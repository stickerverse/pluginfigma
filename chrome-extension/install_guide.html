Image Analyzer Chrome Extension - Installation Guide

This guide will help you install and test the Image Analyzer Chrome extension.

Step 1: Load the Extension

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" using the toggle in the top right
  3. Click "Load unpacked" and select the chrome-extension folder
  4. The extension should now appear in your extensions list
Note: If you see an error about missing icons, you can ignore it. The extension will still function without them.

Step 2: Verify the Extension Is Working

  1. Look for the Image Analyzer icon in your Chrome toolbar (you may need to click the puzzle piece icon to see all extensions)
  2. Click on the Image Analyzer icon to open the popup
  3. You should see the extension interface with "Analyze Images on Page" and "Upload an Image" buttons

Step 3: Test the Extension

  1. Open the included test.html file in Chrome
  2. Click on the extension icon
  3. Click "Analyze Images on Page"
  4. Select one of the sample images to analyze
  5. Alternatively, click "Upload an Image" to test with your own UI mockup
Important: Make sure you've added your Google Cloud Vision API key in the config.js file. The extension will not work without a valid API key.

Step 4: Review the Results

After analyzing an image, you should see:

  • Detected UI components (buttons, text fields, etc.)
  • Extracted color palette
  • Identified text elements and their properties

Troubleshooting

  • If the extension doesn't appear to load, check for any errors in the Chrome Extensions page
  • If image analysis fails, verify your API key is correct and that you have billing enabled for the Vision API
  • Use Chrome DevTools to check for console errors by right-clicking the extension popup and selecting "Inspect"
